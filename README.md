# Beer E-Commerce Challenge

A mobile-first beer e-commerce web application built with React and Express.js. The project consists of a Product Listing Page (PLP) and a Product Details Page (PDP) with real-time stock and price updates.

## Project Structure

```
├── backend/    Express.js API server
├── frontend/   React SPA (Vite)
```

Each folder is a standalone project with its own `package.json` and setup instructions:

- **[Backend README](./backend/README.md)** — API endpoints, installation, and usage
- **[Frontend README](./frontend/README.md)** — Pages, components, styling, and usage

## Quick Start

**1. Start the backend:**

```bash
cd backend
npm install
npm run dev
```

**2. Start the frontend (in a separate terminal):**

```bash
cd frontend
npm install
npm run dev
```

**3. Open** `http://localhost:5173` **in your browser.**

## Tech Stack

- **Frontend**: React, React Router, SASS, CSS Modules, Vite
- **Backend**: Node.js, Express.js
- **Font**: DM Sans
